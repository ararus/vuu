# List Hooks

Three hooks are provided for `List` management, they can be used individually or in combination
to achieve different patterns of behaviour. Whe used in combination, they are most effective when 
combined in the sequence described below. The hooks can be used to override standard `List` behaviour
and to create new, custom components based on `List` primitives. This includes pairing `List` displays
with `Input` or other *focus* elements to create new composite controls such as `Select`, `Combobox`,
`Dropdown` etc. 

- useSelection
- useKeyboardNavigation
- useDropdown
